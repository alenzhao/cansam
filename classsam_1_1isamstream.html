static void sam::samstream_base::initial_exceptions ( iostate  except  )  [static, inherited]

By default, each newly-constructed SAM/BAM stream object has an exceptions mask of failbit|badbit, so throws exceptions on all formatting errors and I/O errors, including failure to open files. (This differs from standard iostreams, which throw no exceptions by default.)

This function can be used to change this default for all subsequently-constructed stream objects. (Use the usual exceptions() method to change an individual stream's exception mask.)

isamstream& sam::isamstream::operator>> ( alignment &  aln  ) 

Similarly to std::istream's extraction operators, this reads one alignment record into aln and returns the stream. The iostate flags will be set at EOF or upon errors, and exceptions will be thrown accordingly as selected via exceptions().

isamstream& sam::isamstream::operator>> ( collection &  headers  ) 

Similarly to std::istream's extraction operators, this reads all header records into headers and returns the stream. The iostate flags will be set at EOF or upon errors, and exceptions will be thrown accordingly as selected via exceptions().

Header records appear at the start of a file, so this must be the first extraction operator used.

The stream retains a reference to headers and hands it to subsequently-extracted alignment records; therefore headers's lifetime must end after the lifetimes of the stream and any resulting alignment objects.
