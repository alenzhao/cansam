Detailed Description

Dereferencing a sam::header iterator or const_iterator produces (a reference to) an instance of this class.

Note:
There are no mutator methods, even if it is a mutable iterator that has been dereferenced; use sam::header::set_field() to change the value of a header field via an iterator.
