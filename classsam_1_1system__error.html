Detailed Description

The message returned by what() is in the format 

message [[for] "filename"]: system error text

where message is as passed to the constructor, the filename portion appears if there is an associated filename (with "for" unless message ends with a space), and the system error text is as returned by strerror(3).

Note:
Because the standard library's errno is a macro, this class's member function can't be named errno().
