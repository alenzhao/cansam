Detailed Description

New code can be added to samsort simply by adding your object file to SAMSORT_OBJS in the makefile. Your source file should include a static object informing the existing code of your new comparison function, like this:

bool less_than(const sam::alignment& a, const sam::alignment& b) {
  // ...
}

static const alignment_comparator
  my_comparator("custom", "Order by my custom criteria", less_than);

This object's constructor runs before main() to add your function to the list; there is no need to change the existing samsort code itself to inform it of your new function.
