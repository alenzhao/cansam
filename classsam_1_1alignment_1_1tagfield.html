Detailed Description

Dereferencing a sam::alignment iterator or const_iterator produces (a reference to) an instance of this class.

Note:
There are no mutator methods, even if it is a mutable iterator that has been dereferenced; use sam::alignment::set_aux() to change the value of an auxiliary field via an iterator.
