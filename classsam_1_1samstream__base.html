Detailed Description

This is a base class for the SAM/BAM stream hierarchy; it cannot be instantiated itself.

Because it is not just characters being transferred, there is buffering and other state in the [io]samstream object as well as the associated streambuf. Hence using the inherited rdbuf(streambuf*) method to change the associated stream buffer has an undefined effect.

Note:
This hierarchy doesn't provide a stream in the sense that anything can be streamed to/from it; it only accepts SAM headers and alignment records. The name does not parallel fstream and stringstream; it does not mean "a stream backed by a sam", whatever that might be.

Member Function Documentation

static void sam::samstream_base::initial_exceptions ( iostate  except  )  [static]

By default, each newly-constructed SAM/BAM stream object has an exceptions mask of failbit|badbit, so throws exceptions on all formatting errors and I/O errors, including failure to open files. (This differs from standard iostreams, which throw no exceptions by default.)

This function can be used to change this default for all subsequently-constructed stream objects. (Use the usual exceptions() method to change an individual stream's exception mask.)
