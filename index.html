C++ binding for SAM/BAM files

The Sequence Alignment/Map (SAM) file format and its binary companion BAM are formats for storing nucleotide sequence alignments. They are designed for the large volumes of short- to middling-length reads produced by next-generation sequencing technologies, and are described at http://samtools.sourceforge.net/ and elsewhere.

Cansam is a C++ library providing classes representing the records found in these files, facilities for reading and writing SAM and BAM files, and several algorithms for manipulating these files, including sorting and merging.

The main things to know about are sam::alignment and sam::header.

Utilities

Several basic command-line utilities are provided along with the library. They are intended to be of use in their own right, as well as providing non-trivial examples of programming with the library.

  • samcat(1) for converting between SAM and BAM formats
  • samhead(1) for displaying headers
  • samsort(1) and samgroupbyname(1) for sorting and merging files
  • samsplit(1) for partitioning files
